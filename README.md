### ローカル実行方法

PyScript は (pyscript.net から配信されているソースを参照する場合は) ローカルの HTML ファイルをクリックしてブラウザで表示しても動作しません。

手元に Python3 環境がある場合は Python3 で Web サーバをたてることができます。このリポジトリを clone したディレクトリで `python -m http.server 8080` と実行した上で http://localhost:8080/ にアクセスしてください（というか HTML ファイルを直接クリックすると PyScript の [Issue  #257](https://github.com/pyscript/pyscript/issues/257#issuecomment-1119595062) からこの方法に誘導されます）。
